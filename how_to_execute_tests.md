## How To Test This

In order to execute this automated tests, you need to:

- First follow the instructions in the [setup.md][Setup.md] file
to execute the Yarn configuration and React App.

- After that, you'll have to configure the Virtual Environment for Python

-- Please make sure you have Python3 version installed.

-- On your python console (or Unix/Linux console), you'll have to run the 

```source /automata/bin/activate
pip install
```

-- On Windows, you'll have to run the following

```/automata/bin/activate.bat
pip install
```

- Then, run the pytests included by executing the following inside the tests folder:

```pytests```
